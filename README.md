# Maahi Foundation Landing Page

A modern, responsive, static NGO/charity landing page built using **HTML5, CSS3, and Vanilla JavaScript**.

## Features

- Responsive NGO-style landing page
- Sticky navigation bar
- Mobile hamburger menu
- Smooth scrolling navigation
- Hero section with Hindi content
- About, Vision, Mission, Projects, Completed Work, Support Us, and Contact sections
- Card-based project layout
- Section reveal animations on scroll
- Font Awesome social icons
- SEO-friendly HTML structure

## Project Structure

```bash
maahi-foundation/

├── js/
```

## How to Run Locally

1. Download or clone this repository.
2. Open `index.html` directly in your browser.

## How to Host on Hostinger

### Option 1: Hostinger File Manager

1. Log in to your Hostinger hPanel.
2. Open **File Manager**.
3. Go to `public_html/` (or your domain’s root folder).
4. Upload all files and folders:
   - `index.html`
   - `css/`
   - `js/`
   - `images/`
5. Ensure `index.html` is in the root (`public_html/index.html`).
6. Visit your domain to verify the site is live.

### Option 2: Upload via FTP (FileZilla)

1. Create/get FTP credentials from Hostinger.
2. Open FileZilla and connect using Host, Username, Password, Port.
3. In remote panel, open `public_html/`.
4. Drag and drop project files from local machine to `public_html/`.
5. Wait for upload completion.
6. Open your domain and test all sections.
